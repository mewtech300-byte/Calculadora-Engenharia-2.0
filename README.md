# Calculadora de Engenharia 2.0

Este projeto faz parte da minha transição de carreira para a área de Tecnologia e Dados. É uma ferramenta desenvolvida em Python para automatizar cálculos complexos utilizados na engenharia.

##  Status do Projeto: Dia 1
Hoje concluí a primeira etapa fundamental do desenvolvimento:
- Configuração do ambiente de desenvolvimento.
- Integração e versionamento com GitHub.
- Estrutura inicial do motor de cálculo (usando bibliotecas como SymPy/Math).

##  Tecnologias que serão  Utilizadas
- **Linguagem:** Python
- **Interface:** Tkinter 
- **Cálculos:** SymPy / NumPy
- **Versionamento:** Git

## Evolução
O Objetivo  com este projeto é aplicar conceitos de lógica de programação e interface gráfica, evoluindo para futuras integrações com análise de dados.
